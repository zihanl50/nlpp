# 淘宝电商 NLP 实战（第 1 周）

## 项目简介
本实验针对淘宝商品标题进行中文文本预处理与分析，验证电商文本的典型特征（短文本、实体密集、噪声多），并完成数据清洗、分词、统计分析及自定义词典效果对比。

## 环境与依赖
- Python 3.8+
- 主要库：pandas, numpy, jieba, matplotlib, seaborn, wordcloud

安装命令：
pip install pandas numpy jieba matplotlib seaborn wordcloud

## 数据来源
阿里天池公开数据集「淘宝商品描述数据集」中的 `content_tag_dataset.txt`，抽取前 10,000 条商品标题进行分析。、

## 主要结果
- **标题长度**：平均约 20 字，符合短文本特征。
- **实体密集性**：品牌词（小米、华为、苹果）和规格词（手机、新款、包邮）出现频次高。
- **噪声分布**：营销词占比约 12%，错别字极少，少量表情符号。
- **自定义词典**：添加“双十一”“绝绝子”后分词准确率明显提升。

## 遇到的问题与解决方案
| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 文本列为空 | 原始数据以 `\t` 分隔，有空行 | 使用 `split('\t')` 并过滤空行 |
| 计算长度报错 | 存在 NaN 值 | `fillna('')` 填充缺失值 |
| 新词被拆分 | jieba 默认词典未收录 | 加载自定义词典（如“绝绝子”） |
| 词云中文乱码 | 缺少中文字体 | 指定系统字体（如 SimHei） |
